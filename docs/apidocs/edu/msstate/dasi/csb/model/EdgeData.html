EdgeData

Related Docs: object EdgeData | package model

case class EdgeData(ts: Long, origPort: Int, respPort: Int, proto: Protocols.Value, duration: Double, origBytes: Long, respBytes: Long, connState: ConnStates.Value, origPkts: Long, origIpBytes: Long, respPkts: Long, respIpBytes: Long) extends Product with Serializable

Defines the edge data which represents the connection properties.

ts

The timestamp of the first packet.

origPort

The originator’s port number.

respPort

The responder’s port number.

proto

The transport layer protocol of the connection.

duration

How long the connection lasted. For 3-way or 4-way connection tear-downs, this will not include the final ACK.

origBytes

The number of payload bytes the originator sent. For TCP this is taken from sequence numbers and might be inaccurate (e.g., due to large connections).

respBytes

The number of payload bytes the responder sent. See origBytes.

connState

The state of the connection.

origPkts

Number of packets that the originator sent.

origIpBytes

Number of IP level bytes that the originator sent (as seen on the wire, taken from the IP total_length header field).

respPkts

Number of packets that the responder sent.

respIpBytes

Number of IP level bytes that the responder sent (as seen on the wire, taken from the IP total_length header field).
