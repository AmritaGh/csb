case class Config(debug: Boolean = false, mode: String = "", partitions: Int = sc.defaultParallelism, seedGraphPrefix: String = "seed", synthGraphPrefix: String = "synth", seedDistributions: String = "seed.distributions", bucketSize: Int = 10, graphLoader: String = "spark", graphSaver: String = "spark", textSaver: String = "none", iterations: Int = 0, connLog: String = "conn.log", synthesizer: String = "", skipProperties: Boolean = false, sampleFraction: Double = 0.1, seedMatrix: String = "seed.mtx", metrics: Seq[String] = Seq("all"), workloads: Seq[String] = Seq("all"), workloadBackend: String = "spark", neo4jUrl: String = "bolt://localhost", neo4jUsername: String = "neo4j", neo4jPassword: String = "neo4j", graphPrefix: String = "graph", srcVertex: VertexId = 0L, dstVertex: VertexId = 0L, patternPrefix: String = "pattern") extends Product with Serializable

Configuration object for the application. Contains all variables relevant to running methods. Not all will apply to every application scenario.

debug

Enables printing of extra command-line output and experimental code.

mode

Specifies the mode that the application should operate in (e.g. seed, synth, veracity).

partitions

Sets number of partitions to use for Spark RDD objects throughout the application.

seedGraphPrefix

Prefix to attach to seed graph when loading/saving it.

synthGraphPrefix

Prefix to attach to synthetic graph when loading/saving it.

seedDistributions

Path to a serialized edu.msstate.dasi.csb.data.distributions.DataDistributions object file.

bucketSize

Size of byte-buckets to break distributions into.

graphLoader

Method to use when loading a graph into memory.

graphSaver

Backend to use when saving a graph.

textSaver

Backend to use when saving a graph as pure-text

iterations

Number of iterations to perform synthesis

connLog

Path to a Bro IDS connection log file, similar to a CSV file.

synthesizer

Method to use when synthesizing a new graph.

skipProperties

Defines whether the application skips over the property-generation sections.

sampleFraction

Percentage to sample during iterations of synthetic graph generation.

seedMatrix

Path to a Kronecker seed matrix file. Saved as an N by N matrix separated by tabs.

metrics

List of metrics to run on both seed and synthetic graphs when in veracity mode.

workloads

List of workloads to run on both seed and synthetic graphs when in workload mode.

workloadBackend

Computation backend for performing workloads.

neo4jUrl

URL string to neo4j instance if using neo4j backend.

neo4jUsername

Username to login to Neo4j with.

neo4jPassword

Password to login to Neo4j with.

graphPrefix

Path prefix of graph to load for workload mode.

srcVertex

Sets a single vertex to use as the source for workload mode.

dstVertex

Sets a single vertex to use as the destination for workload mode.

patternPrefix

Path prefix of the pattern graph to use for subgraph-isomorphism workload.
